# Configuration Directory

This directory contains API keys and configuration files for the Career Guidance app.

## 📁 File Structure

```
lib/config/
├── api_keys.dart           ← Real API keys (GITIGNORED - DO NOT COMMIT)
├── api_keys.sample.dart    ← Template file (safe to commit)
└── README.md              ← This file
```

## 🚀 Quick Setup

### For New Team Members / First-Time Setup

1. **Copy the sample file:**
   ```bash
   cd lib/config
   copy api_keys.sample.dart api_keys.dart
   ```

2. **Edit `api_keys.dart`** and replace placeholder values with your real API keys:
   ```dart
   static const String openAiApiKey = 'sk-proj-YOUR_ACTUAL_KEY_HERE';
   ```

3. **Never commit `api_keys.dart`** to version control!
   - Already in `.gitignore` ✅
   - Git will automatically ignore this file

## 🔐 Security Best Practices

### ✅ DO:
- Keep `api_keys.dart` local only (already gitignored)
- Use `api_keys.sample.dart` as the team template
- Enable API key restrictions in your provider console:
  - IP restrictions
  - Domain restrictions
  - Usage quotas
- Rotate keys regularly
- Use different keys for development and production

### ❌ DON'T:
- Commit `api_keys.dart` to git
- Share keys via email, Slack, or chat
- Hard-code keys directly in other files
- Duplicate Firebase keys here (use `firebase_options.dart` instead)

## 🔥 Firebase Configuration

**Firebase is managed separately!**

- **File:** `lib/firebase_options.dart`
- **Generated by:** FlutterFire CLI
- **Command to regenerate:**
  ```bash
  flutterfire configure
  ```

**Do NOT duplicate Firebase keys in `api_keys.dart`**

## 📝 Adding New API Keys

1. **Update the template** (`api_keys.sample.dart`):
   ```dart
   // Example: Google Maps
   /// Google Maps API key for Android
   /// Get it from: https://console.cloud.google.com/apis/credentials
   static const String googleMapsApiKeyAndroid = 'AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX';
   ```

2. **Update your local file** (`api_keys.dart`) with the real key

3. **Commit only the sample file** to git

## 🛠️ Usage in Code


## ⚠️ Troubleshooting

### "Error: Cannot find `api_keys.dart`"
**Solution:** You need to create it from the sample file (see Quick Setup above)

### "How do I get an OpenAI API key?"
**Solution:** 
1. Go to https://platform.openai.com/api-keys
2. Create an account / sign in
3. Click "Create new secret key"
4. Copy and paste into `api_keys.dart`

### "I accidentally committed `api_keys.dart`!"
**Solution:**
```bash
# Remove from git (but keep local file)
git rm --cached lib/config/api_keys.dart

# Commit the removal
git commit -m "Remove accidentally committed API keys"

# Rotate your API keys immediately!
# Go to your provider's console and regenerate new keys
```
